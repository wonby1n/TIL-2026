# 초보 기준으로 인프라 공부 순서

## 1단계: 리눅스 기본 명령어

```bash
pwd         # 현재 내가 있는 폴더(경로) 보여주기
ls          # 현재 폴더 안에 있는 파일/폴더 목록 보기
cd          # 폴더 이동하기
mkdir       # 새 폴더 만들기
rm          # 파일 또는 폴더 삭제하기
cp          # 파일 또는 폴더 복사하기
mv          # 파일 또는 폴더 이동하기 / 이름 바꾸기
cat         # 파일 내용 한 번에 보기
nano        # 쉬운 텍스트 편집기로 파일 열기
vim         # 리눅스에서 많이 쓰는 편집기로 파일 열기
sudo        # 관리자 권한으로 명령 실행하기
```

추가로 꼭 알아둘 것:

```bash
ps          # 현재 실행 중인 프로세스(작업) 목록 보기
kill        # 실행 중인 프로세스 강제 종료하기
top         # 실시간으로 CPU/메모리 사용량과 프로세스 보기
htop        # top보다 보기 편한 버전의 시스템 모니터
chmod       # 파일/폴더 권한 바꾸기
systemctl   # 리눅스 서비스 시작/중지/상태 확인하기
```

예시:

```bash
pwd                     # 내가 지금 어디 폴더에 있는지 확인
ls                      # 여기 안에 뭐가 들어있는지 확인
cd /home/ubuntu         # /home/ubuntu 폴더로 이동
mkdir myproject         # myproject 폴더 만들기
rm test.txt             # test.txt 파일 삭제
cp a.txt b.txt          # a.txt를 복사해서 b.txt 만들기
mv a.txt folder/        # a.txt를 folder로 이동
cat .env                # .env 파일 내용 보기
nano .env               # .env 파일 수정하기
vim .env                # .env 파일 수정하기(고급 편집기)
sudo apt update         # 관리자 권한으로 패키지 목록 업데이트

ps                      # 실행 중인 작업 보기
kill 1234               # PID가 1234인 작업 종료
top                     # 서버 상태 실시간 확인
htop                    # 보기 편하게 서버 상태 확인
chmod 400 key.pem       # key.pem 권한 변경
systemctl status nginx  # nginx 상태 확인
```

## 2단계: SSH 접속

EC2 들어가는 문

- pem 키 파일
- 서버 주소
- 접속 계정(보통 ubuntu)

이 3개로 접속함.

## 3단계: 포트 개념

서버는 한 대인데, 안에서 여러 프로그램이 돌아갈 수 있어.

이걸 구분하는 문 번호가 포트야.

```
22 = SSH

80 = HTTP

443 = HTTPS

8080 = 백엔드

3000 = 프론트 개발 서버

3306 = MySQL
```

즉,
“서버 주소는 같은데 어떤 프로그램으로 들어갈지”를 포트로 구분해.

## 4단계: 배포


예를 들면:
- 프론트 빌드해서 서버에 올리기
- 백엔드 실행하기
- DB 연결하기
- 환경변수 넣기

Docker는

# 초보가 자주 막히는 포인트

## “환경변수를 잘 모르겠어”

민감한 값 / 서버마다 달라지는 값은 코드 밖으로 빼라


## 1) 권한 문제

- pem 권한
- 파일 수정 권한
- sudo 필요

## 2) 포트 문제

- 서버는 켜졌는데 접속 안 됨
- 보안그룹 / ufw / nginx 문제일 수 있음

## 3) 환경변수 문제

- 값이 빠졌거나 이름이 다름
- .env 읽는 방식이 다름

## 4) 경로 문제

- 지금 내가 어느 폴더인지 모름
- 상대경로/절대경로 헷갈림

## 5) 실행은 됐는데 접속 안 됨

- localhost로만 열려 있음
- 외부 접근 허용 안 됨
