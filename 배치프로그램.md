# 배치 프로그램이란?
## 일괄 처리 (batch processing)
개별적으로 어떤 요청이 있을 때마다 실시간으로 통신하는 것이 아닌 한꺼번에 일괄적으로 대량 건을 처리/ 정해진 특정한 시간에 실행되도록 스케줄 진행
대량 건의 데이터를 처리
특정 시간에 실행
일괄적으로 처리

## 배치 스케줄러

### 스프링 배치 (spring batch)
오픈소스 프레임워크
로그 관리, 추적, 트랜잭션 관리, 작업 처리 통계, 작업 재시작등의 기능 제공

### Quartz
스프링 프레임워크로 개발되는 응용 프로그램들의 일괄 처리를 위한 다양한 기능을 제공하는 오픈 소스 라이브러리

### Cron
리눅스의 스케줄러 도구로 crontab 명령어를 통해 작업 예약

### DB 스케줄러 (오라클 JOB, MYSql 이벤트 스케줄러 등)
이벤트 스케줄러는 mysql/mariadb 서버에서 실행
