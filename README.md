# TIL 2026 🚀

매일 최소 1커밋을 목표로 하는 TIL(Today I Learned) 레포지토리입니다.

## 🔧 규칙

1. 매일 **하루 최소 1개 이상**의 학습/코드 흔적을 남긴다.!
2. 커밋은 의미 있는 변경을 목표로 한다.  
   - 예: 문제 풀이, 개념 정리, 에러 해결, 코드 리팩토링 등
3. 폴더는 **주제 기준**, 파일 이름은 **날짜 + 간단 설명**으로 작성한다.  
   - 예: `2025-11-25_sync-async.md`, `2025-11-25_boj_1764.py`

## 📂 폴더 구조

```text
algorithms/   # 알고리즘, 자료구조, 코테 문제 풀이
web/          # HTML, CSS, JavaScript, API 실습
django/       # Django, DRF 관련 코드 및 실습
notes/        # 강의/개념/이론 정리 (Markdown)
etc/          # 에러로그, 주간 회고 등 기타 기록
